# PrediTrade AI

PrediTrade AI est un assistant d'analyse des marchés financiers conçu pour les débutants.

## Mission

Aider les utilisateurs à comprendre les marchés avant d'investir.

## Fonctionnalités Alpha 1.0

- Création de compte
- Connexion
- Dashboard
- PrediScore™
- Assistant IA
- Historique des analyses

## Vision

Comprendre avant d'investir.
